# 学生积分商城 - UI设计文档

## 1. 文档概述

### 1.1 文档目的
本文档详细描述了学生积分商城系统的用户界面设计方案，包括设计规范、组件设计、页面布局和交互规范，旨在为前端开发提供明确的设计参考和实现指导。

### 1.2 适用范围
本文档适用于学生积分商城系统的前端开发团队，作为UI开发的权威参考。

## 2. 设计系统规范

### 2.1 色彩系统

#### 2.1.1 主色调（蓝色系）
| 名称 | 颜色值 | 用途 |
|------|--------|------|
| Primary 500 | `#1E90FF` | 主要操作按钮、导航高亮 |
| Primary 600 | `#1670D9` | 按钮hover状态 |
| Primary 50 | `#E8F4FF` | 背景装饰、浅色区域 |

#### 2.1.2 辅助色（橙色系）
| 名称 | 颜色值 | 用途 |
|------|--------|------|
| Accent 500 | `#FFA500` | 强调元素、积分显示 |
| Accent 600 | `#E69500` | 橙色按钮hover状态 |
| Accent 50 | `#FFF5E6` | 橙色浅背景 |

| Error | `#FF4D4F` | 错误状态、删除操作 |
